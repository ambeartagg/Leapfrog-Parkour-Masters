# Leapfrog-Parkour-Masters
 First of all, I tried to use the script provided for movement. I altered it to try and make it work better, but ended up somehow making things worse.
 I tried to use copilot to help but still, it didn't work and my players were rolling on the floor.
 Luckily Mae helped me. I deleted the script and started over. I established variables at the beginning, and used a bunch of if statements to choose different movements for players 1 and 2.
 I used the project seetings and changed which keys on my keyboard did what. I made it so the first player has wasd keys for movement and f for jumping. Meanwhile the second player has ijkl for movement, and h is for jumping.
My initial level design was not very fun, so I made it better by created a parkour environment with different platforms to jump on and play with.
